Projects

Alcohol Monitoring and Engine Control System


Objective :

The project aimed to design and develop a safety system to simulate the real-time detection of alcohol levels in a driver and automatically control the engine speed accordingly. The system is intended to prevent accidents caused by drunk driving by regulating the vehicle's engine response based on alcohol concentration detected in the surroundings.

Project Description:

The Alcohol Level Monitoring System utilizes the MQ-3 alcohol sensor, which is capable of detecting alcohol in the air. When the sensor detects alcohol levels above a predefined threshold, the system triggers a mechanism to reduce the speed of the engine. In this simulation, a motor represents the engine, and its speed is reduced proportionally as alcohol levels increase.The system is designed as a prototype using an Arduino board to manage sensor readings and control the motor (engine). The MQ-3 sensor continuously monitors alcohol levels, and based on the readings, it communicates with the motor driver to reduce the engine's speed. This simulates a real-world application where such a system could be implemented in vehicles to enhance road safety by preventing drunk driving.

Technologies Used:
MQ-3 Alcohol Sensor: Detects the concentration of alcohol in the air and sends the data to the Arduino for processing.

Arduino Board: Acts as the main control unit, processing data from the sensor and controlling the motor's speed based on predefined conditions.

Motor Driver: Controls the motor (engine) speed based on signals from the Arduino.

Sensor Interfacing: Integration of the MQ-3 alcohol sensor with the Arduino for real-time monitoring.
